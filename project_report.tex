\section*{4. Tools/Technologies Used}
\begin{itemize}[leftmargin=*, label=--]

  \item \textbf{Frontend:} Streamlit, Plotly, Pandas

\section*{5. SOLID Principles Application}
\textbf{Description:} The codebase applies SOLID principles as follows:
\begin{itemize}[leftmargin=*, label=--]
  \item \textbf{Single Responsibility:} Each module (models, schemas, auth, database, main) has a clear, single purpose.
  \item \textbf{Open/Closed:} New features (e.g., user management, help requests) are added via new endpoints and UI sections without modifying core logic.
  \item \textbf{Liskov Substitution:} Pydantic models and FastAPI dependency injection allow for extensible, substitutable components.
  \item \textbf{Interface Segregation:} Separate schemas for requests and responses, and clear API boundaries.
  \item \textbf{Dependency Inversion:} FastAPI's dependency injection (\texttt{Depends}) is used throughout for database and authentication.
\end{itemize}
